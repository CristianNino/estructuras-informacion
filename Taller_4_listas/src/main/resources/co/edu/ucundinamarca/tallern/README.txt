¿Que son los iteradores en java?
       Un iterador  es un objeto que nos permite recorrer una lista y presentar por pantalla todos sus elementos. para realizar una iteracion se debe tener en 
       cuenta dos metodos los cuales son el hasNext() y  el Next().
 ¿Como podríamos recorrer un colección en java sin la ayuda de in iterador?.
       para recorrer una coleccion en java sin la ayuda de un nterador se usan los ciclos los cuales permitirar buscar o digitar elemento por elemento 
       de una lista hasta que se llegue a su fin.
